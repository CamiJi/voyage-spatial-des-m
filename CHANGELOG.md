# 📋 Changelog

Toutes les modifications notables de ce projet seront documentées dans ce fichier.

Le format est basé sur [Keep a Changelog](https://keepachangelog.com/fr/1.0.0/),
et ce projet adhère au [Semantic Versioning](https://semver.org/lang/fr/).

## [1.0.0] - 2024-01-XX

### ✨ Ajouté
- Système de jeu complet avec difficulté progressive
- Voyage spatial à travers toutes les planètes du système solaire
- Interface de configuration des mots avec article optionnel
- Validation intelligente reconnaissant plusieurs mots possibles
- Animations spatiales et effets visuels
- Thème spatial coloré adapté aux enfants
- Gestion de l'état avec useKV pour la persistance des données
- Support des ligatures françaises (œ, æ)
- Clavier interactif pour la saisie des lettres
- Progression visuelle avec fusée et planètes
- System de cycles avec difficulté croissante

### 🚀 Infrastructure
- Configuration GitHub Actions pour déploiement automatique
- Support GitHub Pages
- Scripts de configuration de déploiement
- Documentation complète (README, DEPLOYMENT, QUICKSTART)
- Templates d'issues pour bugs et fonctionnalités
- Guide de contribution

### 🎨 Design
- Police Fredoka pour une ambiance ludique
- Palette de couleurs spatiales (bleu profond, jaune doré)
- Arrière-plan spatial animé avec étoiles
- Cartes de lettres interactives
- Représentations cartoon des planètes
- Animations fluides avec framer-motion

### 🐛 Corrections
- Fix de la gestion des espaces entre article et mot
- Amélioration de la normalisation du texte pour la validation
- Correction de la progression de difficulté
- Fix du retour à la Terre en cas d'erreur
